# DocSense Web

React + Vite frontend for DocSense.

## What it does
- Firebase auth UI + route guard
- Landing page + basic chat UI scaffold

## Run locally
```bash
npm install
npm run dev
```

## Env
- Copy `.env.example` to `.env.local` (or set vars in your shell)
- Vite exposes only variables prefixed with `VITE_`
